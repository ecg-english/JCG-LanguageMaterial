# 🍽️ Restaurant Ordering & Conversation - 日本語レストラン表現集

日本語学習者のための完全レストラン表現集ウェブアプリケーション

## 📚 概要

このウェブアプリは、日本のレストランで必要な日本語表現を効率的に学習できるように設計されています。入店から会計まで、実践的なフレーズと詳しい解説、練習問題を提供します。

## ✨ 特徴

### 📖 5つの学習セクション
1. **Before Ordering (注文前)** - 入店、予約、席の希望
2. **Ordering (注文)** - メニュー閲覧、注文、アレルギー対応
3. **Mealtime (食事中)** - 食事中の表現、追加注文
4. **Problems & Requests (問題とリクエスト)** - トラブル対応、依頼
5. **Paying the Bill (会計)** - 支払い、領収書、別会計

### 🎯 各セクションの内容
- **実用的なフレーズ** - すぐに使える表現集
- **詳細な解説** - 文法や使い方の丁寧な説明
- **文化的背景** - 日本の食文化やマナーの紹介
- **例文と応用** - 様々な状況に対応できる表現
- **練習問題** - 理解度を確認するクイズ
- **進捗トラッカー** - 学習の進み具合を可視化

### 💡 学習機能
- **インタラクティブなクイズ** - 各セクションに3問の選択式問題
- **即座のフィードバック** - 正解・不正解がすぐにわかる
- **進捗管理** - 全体の学習進捗をパーセンテージで表示
- **わかりやすいUI** - 直感的で美しいデザイン

## 🚀 使い方

### インストール不要
1. `index.html` ファイルをブラウザで開くだけ！
2. インターネット接続不要でオフラインで使用可能

### 推奨ブラウザ
- Google Chrome (最新版)
- Firefox (最新版)
- Safari (最新版)
- Microsoft Edge (最新版)

### 学習の進め方
1. **順番に学習** - セクション1から順番に進むのがおすすめ
2. **フレーズを読む** - 各フレーズの日本語、読み方、英訳、解説を確認
3. **クイズに挑戦** - 各セクションの最後にある練習問題を解く
4. **実践練習** - 実際のレストランで使ってみる！

## 📱 レスポンシブデザイン

スマートフォン、タブレット、PCなど、あらゆるデバイスで快適に学習できます。

## 🎨 主な機能

### 色分けされたカード
- **フレーズカード** - 見やすいグラデーションデザイン
- **文化ノート** - 黄色の背景で文化情報を強調
- **例文ボックス** - 緑色で関連表現を表示
- **使用ノート** - 黄色で重要なポイントを表示

### クイズシステム
- 選択式問題
- 正解時は緑色、不正解時は赤色で表示
- 正解数の自動カウント
- 詳しい解説付き

### 進捗トラッカー
- リアルタイムで学習進捗を表示
- 全15問中の正解数をパーセンテージで表示

## 📖 学習内容の例

### セクション1: Before Ordering
- 人数の伝え方
- 予約の確認
- 席の希望
- 待ち時間の確認

### セクション2: Ordering
- メニューの閲覧
- おすすめの確認
- アレルギーの伝え方
- 量の調整

### セクション3: Mealtime
- 「いただきます」「ごちそうさまでした」
- 料理の感想
- 追加注文
- 食器の依頼

### セクション4: Problems & Requests
- 注文の遅れ
- 間違った料理
- 温度の問題
- 環境調整

### セクション5: Paying the Bill
- 会計の依頼
- 支払い方法
- 別会計
- 領収書

## 🌟 学習のコツ

1. **音読練習** - フレーズを声に出して読む
2. **実践重視** - 実際のレストランで使ってみる
3. **復習** - 定期的に見返して記憶を定着させる
4. **状況を想像** - それぞれの場面を具体的にイメージする
